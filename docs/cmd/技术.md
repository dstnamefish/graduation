# 技术栈
### 后端技术栈
核心框架: Spring Boot 2.7.x
数据库: MySQL 8.0 + MyBatis-Plus
认证授权: Spring Security + JWT
API文档: Knife4j (Swagger增强版)
构建工具: Maven
其他: Lombok, Hutool工具库

### 前端技术栈
核心框架: Vue 3 + Composition API
构建工具: Vite 4.x
UI组件库: Element Plus 
状态管理: Pinia
路由: Vue Router 4
HTTP客户端: Axios
工具库: lodash-es, dayjs

五、工具推荐
VS Code 插件：
Vetur：提供 Vue 模板和 SCSS 的语法高亮
ESLint：强制代码风格和注释规范
Document This：自动生成 JSDoc 注释
SCSS IntelliSense：智能提示 SCSS 变量和混合器
文档生成工具：
TypeDoc：从 TypeScript 注释生成 API 文档
VuePress：生成组件库文档
SassDoc：为 SCSS 生成样式文档
