# npm cmd
```
npm install vue-router@4 --save
```

| 部分           | 作用                                                         |
| :------------- | :----------------------------------------------------------- |
| `npm install`  | Node.js 的包管理工具（npm）的安装命令。                      |
| `vue-router@4` | 指定安装 **Vue Router 4.x 版本**（专为 Vue 3 设计，与 Vue 2 不兼容）。 |
| `--save`       | 将依赖包信息保存到 `package.json` 的 `dependencies` 中（npm 5+ 后可省略）。 |
