# 插件作用介绍（规整版）

1. **Auto Close Tag**：自动添加HTML/XML闭合标签，功能与Visual Studio IDE或Sublime Text一致，减少手动输入闭合标签的操作，避免标签遗漏问题。
2. **Auto Import**：自动查找、解析可用的导入资源，并为TypeScript和TSX提供代码操作建议与代码补全功能，简化导入语句编写流程，提升编码效率。
3. **Auto Rename Tag**：自动重命名成对的HTML/XML标签，当修改起始标签名称时，对应的结束标签会同步更新，无需手动修改，确保标签配对一致性。
4. **Code Spell Checker**：一款针对源代码的拼写检查工具，可检查注释、变量名、函数名等内容中的英文拼写错误，帮助规范代码书写，避免低级拼写问题。
5. **Color Highlight**：在编辑器中高亮显示网页颜色值，无论是CSS中的十六进制色、RGB色还是颜色关键词，都能直观呈现，方便前端开发人员快速识别和调整颜色样式。
6. **Database Client JDBC**：为数据库客户端提供JDBC适配功能，支持通过JDBC驱动连接各类数据库，实现在VS Code中直接操作数据库的需求，无需额外安装专用数据库管理工具。
7. **Debugger for Java**：一款轻量级的Java调试工具，适用于Visual Studio Code，支持设置断点、查看变量值、单步执行等常用调试操作，助力Java程序问题排查与调试。
8. **Devbox**：为sailos/sealos中的云开发箱提供辅助代码支持，涉及云开发环境的配置、调试等相关功能，帮助开发者更便捷地在云开发环境中开展工作。
9. **Element Plus Snippets**：提供Element-Plus组件的代码片段，Element-Plus是基于Vue.js 3.0的Web UI工具包，该插件可快速插入常用组件代码，简化Vue3+Element-Plus项目开发流程。
10. **Error Lens**：增强代码中错误、警告及其他语言诊断信息的高亮显示效果，让问题位置和提示内容更醒目，帮助开发者快速发现并定位代码中的问题。
11. **ESLint**：将ESLint工具集成到VS Code中，实时检查JavaScript和TypeScript代码是否符合预设的语法规则与编码规范，发现潜在错误和不规范写法，并提供修复建议，提升代码质量与可维护性。
12. **Extension Pack for Java**：包含一系列Java开发常用扩展的集合，提供Java智能感知、调试、测试、Maven/Gradle构建支持、项目管理等功能，为Java开发者在VS Code中搭建全面的开发环境。
13. **Extension Pack for Java Auto Config**：自动完成JDK配置，同时集成“Extension Pack for Java”和“Spring Boot Extension Pack”，简化Java开发环境的搭建步骤，帮助开发者快速启动Java项目开发。
14. **GitLens — Git supercharged**：增强VS Code对Git的集成能力，可在代码行旁显示代码作者、最后修改时间等Git提交信息，支持查看文件历史记录、比较不同版本代码差异等操作，助力理解代码演变过程与团队协作。
15. **Gradle for Java**：用于在VS Code中管理Gradle项目，支持运行Gradle构建任务，优化Gradle配置文件的编写体验，为使用Gradle构建工具的Java开发者提供便捷支持。
16. **Java**：Visual Studio Code的Java平台扩展，提供基础的Java语言支持，包括语法高亮、代码补全、基础语法检查等功能，是Java开发的基础插件之一。
17. **Java DebugX**：增强VS Code中的Java调试功能，提供宏录制、运行时洞察、动态方法调用等高级调试特性，帮助开发者更高效地排查Java程序中的复杂问题。
18. **Language Support for Java(TM) by Red Hat**：由Red Hat开发的Java语言支持插件，具备Java代码检查、智能感知、代码格式化、重构以及Maven/Gradle构建支持等功能，全面提升Java开发效率与代码质量。
19. **Live Preview**：在工作区内启动本地服务器，用于预览网页效果，开发者可在编写HTML、CSS、JavaScript代码时，实时查看页面渲染结果，无需频繁切换到外部浏览器。
20. **Live Sass Compiler**：实时将Sass或Scss文件编译为CSS文件，开发者编写样式代码后，插件会自动完成编译并更新CSS文件，实现样式修改的实时生效，提升前端样式开发效率。
21. **Live Server**：启动带有实时重新加载功能的本地开发服务器，支持静态和动态页面预览，当修改HTML、CSS或JavaScript文件并保存后，浏览器会自动刷新页面，实时展示修改效果。
22. **Live Server Preview**：用于预览HTML文件，启用本地服务器的实时重新加载功能，开发者在编辑HTML页面时，可实时查看页面内容变化，简化静态页面开发调试流程。
23. **Lombok Annotations Support for VS Code**：一款轻量级扩展，支持在VS Code中处理Lombok注解，开发者可使用Lombok注解自动生成getter、setter、构造函数等模板代码，简化Java代码编写。
24. **Markdown All in One**：整合编写Markdown所需的各类功能，包括键盘快捷键、目录自动生成、实时预览等，为使用VS Code编写Markdown文档的开发者提供一站式支持。
25. **markdownlint**：对VS Code中的Markdown文件进行语法检查（linting）和样式规范检查，帮助开发者确保Markdown代码的格式与风格符合行业规范，提升文档可读性。
26. **Maven for Java**：用于在VS Code中管理Maven项目，支持执行Maven构建目标、从项目原型生成新项目等操作，优化Maven用户的开发体验，简化Java项目的构建与管理流程。
27. **MySQL**：一款多数据库管理工具，支持MySQL/MariaDB、PostgreSQL、Redis、ElasticSearch等数据库的管理，提供可视化操作界面，支持执行SQL查询、管理数据库对象等功能。
28. **open in browser**：允许开发者在默认浏览器或指定应用程序中打开当前编辑的文件，尤其适用于HTML文件的快速预览，无需手动在浏览器中导航文件路径。
29. **Outline Map**：提供可视化的交互式代码大纲地图，融合了代码大纲的清晰性与缩略图的直观概览功能，是VS Code内置大纲功能的增强版本，方便在长文件中快速定位函数、类等代码结构。
30. **Path Intellisense**：Visual Studio Code的文件路径自动补全插件，在开发者输入文件路径时提供智能提示，减少路径拼写错误，提升文件引用操作的准确性与效率。
31. **Pinia Snippets**：提供Pinia状态管理库的代码片段，Pinia是适用于Vue.js的轻量级状态管理工具，该插件可快速插入常用的Pinia代码模板，简化Vue项目状态管理代码编写。
32. **Prettier - Code formatter**：基于Prettier的代码格式化工具，支持多种编程语言，可根据预设规则自动格式化代码，统一代码风格，使代码更整洁、易读，还可配置为保存文件时自动执行格式化。
33. **Project Manager for Java**：用于在VS Code中管理Java项目，支持创建、打开、切换和管理多个Java项目，提供便捷的项目管理界面，提升多项目开发时的效率。
34. **px to rem & rpx & vw (cssrem)**：在VS Code中实现px与rem、rpx、vw单位之间的转换，支持一键转换或实时计算，方便前端开发人员根据设计需求和不同设备尺寸调整样式单位。
35. **React Developer Tools**：提供实用的React开发辅助工具，可加快React组件创建、React文件生成等操作流程，为React开发者提供便捷支持，提升React项目开发效率。
36. **Remote - SSH**：支持通过SSH协议在远程机器上打开任意文件夹，并可充分利用VS Code的全部功能，实现本地编辑器操作远程服务器代码的需求，简化远程开发流程。
37. **Remote - SSH: Editing Configuration Files**：专门用于编辑SSH配置文件的插件，提供配置项智能提示与语法高亮，方便开发者配置远程服务器连接参数（如主机地址、用户名、端口等）。
38. **Remote Explorer**：用于查看远程机器的SSH连接和隧道连接列表，提供可视化的远程环境管理界面，方便开发者快速连接和切换远程开发环境。
39. **SCSS Intellisense**：为SCSS样式文件提供高级自动补全和代码重构支持，包括变量、混合器、函数等SCSS特性的智能提示，帮助开发者更高效地编写SCSS代码。
40. **Simple React Snippets**：提供简洁实用的React代码片段，开发者可通过输入简短的快捷命令生成常用的React组件代码模板，减少重复编码工作，提升React开发效率。
41. **Spring Boot Dashboard**：VS Code的Spring Boot仪表盘插件，支持启动、停止、重启Spring Boot应用程序，实时查看应用运行状态，为Spring Boot项目的管理与监控提供便捷支持。
42. **Spring Boot Extension Pack**：包含一系列用于开发Spring Boot应用程序的扩展集合，提供项目创建、配置文件智能提示、调试支持等功能，为Spring Boot开发提供全面保障。
43. **Spring Boot Tools**：为Spring Boot的application.properties和application.yml配置文件提供语法验证与内容补全支持，同时为Java文件提供Spring Boot专属功能支持，助力Spring Boot项目高效开发。
44. **Spring Initializr Java Support**：基于Spring Initializr的轻量级扩展，支持快速生成Spring Boot Java项目骨架，开发者可通过插件选择项目依赖、版本等配置，简化新项目初始化流程。
45. **SQLTools**：一款多数据库连接工具，支持连接多种常用数据库，提供便捷的数据库管理界面，支持执行SQL查询、管理数据库表、视图等对象，满足开发者在VS Code中操作数据库的需求。
46. **Stylelint**：VS Code的官方Stylelint扩展，用于检查CSS和SCSS代码的风格规范，预防样式代码中的潜在错误，确保样式编写符合团队或行业规范。
47. **Test Runner for Java**：支持在VS Code中运行和调试JUnit或TestNG测试用例，提供可视化的测试结果展示，帮助Java开发者便捷地开展单元测试工作，确保代码质量。
48. **Thunder Client**：一款轻量级的REST API客户端，集成于VS Code中，支持发送HTTP请求、查看响应结果、保存请求历史等功能，无需额外安装Postman等工具，简化API调试流程。
49. **Todo Tree**：在项目中搜索包含TODO、FIXME等注释标记的代码行，并在侧边栏以树状结构展示，方便开发者快速定位和管理待办事项与需要修复的问题，提升项目管理效率。
50. **Trailing Spaces**：高亮显示代码中的尾随空格（行尾多余空格），并支持一键删除所有尾随空格，帮助开发者保持代码整洁，避免因多余空格导致的格式问题。
51. **Turbo Console Log**：自动在JavaScript和TypeScript代码中插入、管理和清理日志信息，基于AST（抽象语法树）确保日志插入精准，支持控制台日志或自定义日志函数，简化调试过程中的日志管理。
52. **Vite**：为Vite构建工具提供VS Code支持，包括Vite项目的语法高亮、代码补全、运行与调试集成等功能，助力开发者在VS Code中高效开展Vite驱动的前端项目开发。
53. **Volar Labs**：与Vite配套的扩展，Volar是Vue.js开发的核心支持插件，Volar Labs提供一些实验性或高级功能，进一步增强Vue项目的开发体验，探索Vue开发的新特性。
54. **Vue (Official)**：Vue.js官方推出的语言支持插件，提供对Vue文件的语法高亮、智能提示、代码补全、错误校验等功能，为Vue开发者提供官方级别的开发支持，确保Vue代码编写的规范性与高效性。
55. **Vue 3 Snippets**：提供Vue.js 3和Vue.js 2的代码片段，开发者可通过简短的快捷命令生成常用的Vue组件代码模板，适配不同版本Vue项目的开发需求，提升Vue代码编写效率。
56. **XML**：由Red Hat开发的XML语言支持插件，提供XML文件的语法高亮、智能感知、代码格式化等功能，帮助开发者便捷地编辑XML配置文件或数据文件，确保XML代码的规范性。
