### rem 和 px 区别

`#前端样式` `2025-07-15` 

- rem 和 px 是前端开发中常用的长度单位。
- **px（像素）**：绝对单位，表示屏幕上的一个物理像素点，尺寸固定，适合精确布局，但不利于响应式设计。
- **rem（root em）**：相对单位，基于根元素（html）的 font-size，随根元素字体大小变化，适合响应式和移动端开发，便于全局缩放和适配。
- **应用场景**：移动端和响应式页面推荐用 rem，需精确像素控制时用 px。
- **示例**：

  ```css
  html {
    font-size: 16px;
  }
  .box1 {
    width: 32px;
  } /* px 固定宽度 */
  .box2 {
    width: 2rem;
  } /* rem 随 html 字体大小变化 */
  ```
